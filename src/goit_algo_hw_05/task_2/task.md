# Task 2

Реалізуйте двійковий пошук для відсортованого масиву з дробовими числами.

Написана функція для двійкового пошуку повинна повертати кортеж:

- перший елемент
  - кількість ітерацій, потрібних для знаходження елемента
- другий елемент
  - "верхня межа". Це найменший елемент, який є більшим або рівним заданому значенню.
